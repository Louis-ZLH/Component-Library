<Section id={1}>
  ## <WithHighlightColor>Props Explanation</WithHighlightColor>
  The Button component accepts a variety of props to customize its appearance
  and behavior: - `variant`: Defines the visual style of the button (e.g.,
  "primary", "secondary", "glass"). - `size`: Specifies the size of the button
  (e.g., "small", "medium", "large"). - `disabled`: A boolean that, when true,
  disables the button and applies appropriate styles. - `onClick`: A function
  that is called when the button is clicked. - `children`: The content to be
  displayed inside the button, typically text or icons.
</Section>
