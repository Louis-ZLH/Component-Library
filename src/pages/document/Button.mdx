<Section id={4}>
  ## Theming

The theme of Button component now supports light mode and dark
mode. To switch between light mode and dark mode, you need to add `dark`
class to the container itself or to the `<html>` elements.

`<html>` elements is recommended since it can cover the whole page.

  <DemoBlock
    code={`import { Button } from "component-lib-core";
import ChangeThemeButton from "unknown";
export default function ButtonThemeDemo() {
  return (
    <div className="flex flex-wrap items-center gap-2">
      <Button variant="glass">Theme</Button>
      <ChangeThemeButton />
    </div>
  );
}`
    }
  >
  <div className="flex flex-wrap items-center gap-2">
    <Button variant="glass">Theme</Button>
    <Button variant="primary">Theme</Button>
    <ChangeThemeButton />
  </div>
  </DemoBlock>

<Section id={5}>
  ## Customization
  You can customize the Button component using your own styles. Here is an example of creating a glass-style button by applying custom styles.

Below is a example of customizing the glass button elegantly with red tint and smooth transition effect:

<DemoBlock
  code={`
import { Button } from "component-lib-core";
export default function CustomButtonDemo() {
  return (
    <Button variant="glass" className={"bg-red-400/10 hover:bg-red-500/20 trainsition duration-500 dark:bg-[color-mix(in_srgb,#fca5a5_40%,#737373_70%)]/50 dark:hover:bg-[color-mix(in_srgb,#fca5a5_70%,#525252_30%)]/70"}>
      Customized Button
    </Button>
  );
}
    `}
>
  <Button
    variant="glass"
    className={
      "trainsition bg-red-400/10 duration-500 hover:bg-red-500/20 dark:bg-[color-mix(in_srgb,#fca5a5_40%,#737373_70%)]/50 dark:hover:bg-[color-mix(in_srgb,#fca5a5_70%,#525252_30%)]/70"
    }
  >
    <>Customized Button</>
  </Button>
</DemoBlock>

For further customization, please refer to the <LinkButton href="/docs/hoc"><Strong>Higher Order Component</Strong></LinkButton>.
